# Package index

## Phylogeny and trace plots

Plot phylogenetic trees alongside per-patient location traces, the
headline figures of the package.

- [`plot_trace_phylo_tree()`](https://theabhirath.github.io/hospitraceRVisualize/reference/plot_trace_phylo_tree.md)
  : Plots a phylogeny with trace heatmap using ggtree with consistent
  sizing
- [`plot_trace_phylo_legend()`](https://theabhirath.github.io/hospitraceRVisualize/reference/plot_trace_phylo_legend.md)
  : Create standalone legend for trace phylogeny plots
- [`plot_trace_phylo_presence()`](https://theabhirath.github.io/hospitraceRVisualize/reference/plot_trace_phylo_presence.md)
  : Plots a phylogeny with presence/absence trace heatmap and
  surveillance cell overlay
- [`plot_clusters_phylo()`](https://theabhirath.github.io/hospitraceRVisualize/reference/plot_clusters_phylo.md)
  : Maps clusters onto a phylogenetic tree and visualizes them

## Cluster comparison and overlap plots

Visualize comparisons between cluster assignments and epidemiological
overlap analyses.

- [`plot_jaccard_similarity_heatmap()`](https://theabhirath.github.io/hospitraceRVisualize/reference/plot_jaccard_similarity_heatmap.md)
  : Plots a heatmap of the Jaccard similarity between clusters created
  with two different methods.
- [`plot_overlap_perm_test()`](https://theabhirath.github.io/hospitraceRVisualize/reference/plot_overlap_perm_test.md)
  : Plot overlap permutation test results as violin plots

## Cluster statistics plots

Summarize genetic variation within and between transmission clusters.

- [`plot_intra_vs_inter_cluster_distance()`](https://theabhirath.github.io/hospitraceRVisualize/reference/plot_intra_vs_inter_cluster_distance.md)
  : Plot max intra-cluster vs. min inter-cluster genetic distance.
- [`plot_intra_vs_inter_isolate_distance()`](https://theabhirath.github.io/hospitraceRVisualize/reference/plot_intra_vs_inter_isolate_distance.md)
  : Plot max intra-cluster vs. min inter-isolate genetic distance.
